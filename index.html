# Супервайзер Достижения Бекойны Действия

📆 Недельное начисление баллов

📋 Правила начисления баллов

Баллы (БЕКОйны) начисляются и списываются по следующим правилам:

📊 Недельное начисление

По 5 категориям:

  • Рейтинг
  • Повышение места
  • Понижение места
  • Best practice
  • Фокусные позиции

По каждой категории действует правило:

МестоБонус
1-е+10 баллов
2-е+7 баллов
3-е+4 балла
4-е+1 балл

Баллы за понижение места вычитаются по тому же принципу: количество потерянных мест = количество вычитаемых баллов.

🏅 Достижения (ачивки)

  • 🏅 План – при достижении 100 баллов
  • 📚 Эксперт – при 200 баллах
  • ⚡ Красавчик – при 300 баллах
  • 🌟 Лидер – при 400 баллах
  • 💎 Мастер – при 500 баллах

Все изменения автоматически сохраняются в Google Таблицу и синхронизируются между пользователями.
